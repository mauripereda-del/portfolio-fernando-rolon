# Portfólio Pessoal — Fernando Rolon

Currículo online desenvolvido com **HTML5**, **CSS3** e **JavaScript**

**Matrícula:** RU5480032

## Páginas

| Página | Arquivo | Descrição |
|--------|---------|-----------|
| Sobre mim | `index.html` | Apresentação pessoal e hobbies |
| Formação | `formacao.html` | Cursos, idiomas e formação acadêmica |
| Portfólio | `portfolio.html` | Links para projetos desenvolvidos |
| Contato | `contato.html` | Formulário com validação JavaScript |

## Funcionalidades JavaScript

- Menu responsivo (hambúrguer) em dispositivos móveis
- Alternância entre tema claro e escuro (salvo no navegador)
- Validação do formulário de contato (campos obrigatórios + e-mail válido)
- Simulação de envio com modal de confirmação
- Filtro de projetos por categoria na página Portfólio
